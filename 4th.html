HTML TAGS

  1. "doctype": Defines the document type
  2. "h1 to h6" : Defines HTML headings
  3. "head" : Contains metadata/information for the document
  4. "header" : Defines a header for a document or section
  5. "hr" : Defines a thematic change in the content
  6. "html" : Defines the root of an HTML document
  7. "i" : Defines a part of text in an alternate voice or mood
  8. "iframe" : Defines an inline frame
  9. "img" : Defines an image
  10. "input" : Defines an input control
  11. "ins" : Defines a text that has been inserted into a document
  12. "a": Defines a hyperlink
  13. "abbr": Defines an abbreviation or an acronym
  14. "address": Defines contact information for the author/owner of a document
  15. "area" : Defines an area inside an image map
  16. "article" :Defines an article
  17. "body" : Defines the document's body
  18. "div": Defines a section in a document
  19. "em" : Defines emphasized text
  20. "dl" : Defines a description list
  21. "dt" : Defines a term/name in a description list
